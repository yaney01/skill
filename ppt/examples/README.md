# HTML PPT examples

These examples are regression fixtures as well as design references. Each example must use the canonical runtime under `../assets/runtime/`, pass structural validation in development form, bundle into one portable HTML file, and pass rendered browser QA.

## Included examples

- [`ai-ad-workflow`](./ai-ad-workflow/) — 12-page Chinese speaker-led deck using a Swiss Grid × Technical Field visual system. It covers cover, statement, section, data hero, comparison, image split, three-column, evidence grid, process, quote, timeline, and closing layouts.

Generated `dist/` files and screenshot folders are intentionally not committed. Rebuild them through the scripts in the example README.
